Description

This application lets you manage your car inventory.

Technologies used

Angular 7 REST API PHP Code Igniter  Mysql

This application has three modules including -

Inventory
- Here you can manage your inventory. Data listed in the grid is grouped based on the manufacturer and model.
Add Manufacturer
- This lets you add a manufacturer. The manufacturer name must be unique.
Add Model
- Here you can add the different vehicle models.
